## [2.0.3] - 2026-04-22

### 🛠️ 基础设施
- 新增 GitHub Actions CI，覆盖 lint、类型检查、单元测试覆盖率、构建和 Playwright E2E。
- CI 使用 `actions/checkout@v5`、`actions/setup-node@v5`、`actions/upload-artifact@v5`，并显式启用 Node.js 24 action runtime，避免新增流水线带入 Node.js 20 runtime 弃用风险。
- 新增 Vitest 覆盖率报告与阈值检查，并提供稳定的 `npm run test:coverage` 命令。
- 新增 Next.js ESLint 配置，避免 CI 中 `next lint` 进入交互式初始化。
- 为 Dexie `pendingDeletions` 增加 `[tableName+recordId]` 复合索引，消除同步删除查询性能警告。

### 🔎 可观测性
- 新增轻量前端错误监控，自动捕获未处理异常、Promise 拒绝和同步/API 错误。
- 新增 `/api/client-errors` 同域错误上报端点，默认写入脱敏后的服务端结构化日志。
- 上报链路新增 URL 与上下文脱敏，避免密码重置 token、邮箱、密码和授权信息进入日志。

### ✅ 验证
- `npm run lint`
- `npm run typecheck`
- `npm run test:coverage`
- `npm run build`
- `npm run test:e2e`

## [2.0.0] - 2026-04-21

### ✨ 新增
- 共享空间同步升级为 `SSE 事件通知 + 增量拉取`，并补充增量游标、连接状态和本地冲突表。
- 设置页新增同步状态看板与冲突决议入口；变更记录页补充成员和表类型过滤。
- 图片改为本地文件存储，新增菜单图片上传/读取/删除 API，并补齐对 legacy base64 图片的服务端与客户端迁移。
- 个人备份升级为 ZIP 完整备份包，导出 `backup.json + images/`，同时保持旧 JSON 备份导入兼容。
- 新增收藏、推荐、场景清单和评论撤回能力；首页、随机页与详情弹窗补充对应入口。
- 账号体系正式切换为 `邮箱 + 密码`；新增注册、登录、修改密码、忘记密码、旧本机身份绑定，以及基于 QQ SMTP 的重置密码邮件。
- 所有 `/api/sync/*` 和变更记录读取接口统一收口到“登录态 + 空间成员身份”校验，客户端不再可信任 `profile_id / space_id`。

### 🐛 修复
- 收口个人备份语义；现在只导出本地私有数据，并补齐界面设置与忌口备份，同时排除共享空间同步来的菜单、标签、模板和互动数据。
- 收口共享空间下的重置与导入行为；重置会清空当前设备本地缓存后重新同步当前空间，个人备份导入则仅允许在本地模式下执行。
- 修复保留共享身份但清空本地库时仍会自动注入示例数据的问题，避免示例数据与共享空间回拉数据混在同一会话里。
- 修复共享同步接口裸露、旧匿名身份可被任意认领，以及生产环境下邮箱验证码登录不可用的问题；旧身份迁移现在只允许绑定当前设备本地保存的旧身份。

### ✅ 验证
- `npm run test:run`
- `npm run test:e2e`
- `npx tsc --noEmit`
- `npm run build`

## [1.0.0] - 2025-04-13

### 🎉 初始版本发布

> 从今天起，不再纠结"今天吃什么"。

### ✨ 功能
- **随机抽选**：支持"自己做"、"叫外卖"和自定义组合模板抽选。
- **菜单管理**：添加 / 编辑 / 删除菜品，支持标签分类、材料清单、烹饪步骤和图片上传。
- **标签体系**：自定义标签（菜系、类别、自定义），支持多标签筛选与搜索。
- **组合模板**：按规则批量抽取，如"1 荤 + 1 素 + 1 汤"。
- **历史记录**：自动记录每次抽选结果，随时回看"今天吃了什么"。
- **忌口管理**：标记忌口菜品，抽选时自动规避，支持一键查看与取消。
- **多设备同步（初始形态）**：早期版本通过邀请码加入同一共享空间，以轮询方式准实时同步菜单数据；后续版本已升级为账号登录与更严格的共享鉴权。
- **PWA 支持**：可安装到手机桌面，离线可用，已针对移动端进行交互和布局优化。

### 🛠️ 技术
- 前端：Next.js 14 (App Router) + React 18 + TypeScript + Tailwind CSS
- 后端：Next.js API Routes + better-sqlite3（单文件 SQLite）
- 客户端存储：Dexie.js (IndexedDB)
- 测试：Vitest + jsdom，47 项单元测试全部通过

### 🐛 修复
- 修复种子数据缺失 `spaceId` 导致的编辑后无法同步问题。
- 修复移动端弹窗按钮和标签区域可能溢出的问题。
- 修复菜单页 React 异步状态更新警告。
